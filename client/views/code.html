Example 1: HTML Embedded Image Slideshow

The image gallery with quotes overlaid on the home page of this site was not easy to produce. CSS3 does provide an API for animations and transitions, but 'background' and its child properties are not within the scope of that functionality.

First, let's consider that the slideshow lives in a container like this: 

<div>
    <!-- Slideshow Here -->
</div>


This is intuitive, what we have to decide now is how to properly create the desired dimensions, but in my case (on this site) the slideshow is actually rendered within '<div ng-view></div>', which is nested within a few other elements: 
<body ng-app="myHomepage" ng-cloak layout="column">
  <vaughn-toolbar></vaughn-toolbar>
  <md-content class="main">
    <div ng-view></div>
    <br><hr md-colors="{borderColor: 'primary-600', backgroundColor: 'primary-600'}">
  </md-content>
  <vaughn-footer hide show-gt-xs></vaughn-footer>
</body>


If this seems foreign, I am using Angular.js and Angular-Material directives. Let's break it down:

Custom Directive: <vaughn-toolbar><vaughn-toolbar> 

<md-toolbar md-whiteframe="3" style="overflow-x: auto; padding: 0.1em">
  <div class="md-toolbar-tools" md-colors="{color: 'primary-50'}">

    <div ng-repeat="vaughnButton in beforeText">
      <vaughn-button-inner></vaughn-button-inner>
    </div>

    <span flex></span>
      <h2 hide show-gt-sm>Welcome to Vaughn's Playground</h2>
    <span flex></span>

    <div ng-repeat="vaughnButton in afterText">
      <vaughn-button-inner></vaughn-button-inner>
    </div>

</div>
</md-toolbar>
